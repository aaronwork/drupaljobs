Welcome to Drop Jobs!
---------------------

This folder and all of its subfolders contain code for the Drop Jobs Drupal
distribution.

For up-to-date documentation on the distribution please check out the resources
on the project's official page: http://drupal.org/project/drop_jobs and the
official user group: http://groups.drupal.org/drop-jobs-distribution

Thanks for using Drop Jobs!

Developers
----------

In order to help out with Drop Jobs all you have to do is build the development
version.

It's as simple as running "./build.sh" (you might need to give it u+x permission)
and following the instructions on-screen.

Thanks!!!
